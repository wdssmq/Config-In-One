## Git 使用 symlinks 的正确姿势

- 安装时开启下边选项

  ![001](doc/001.png "001")

- `.git/config` 中开启「core.symlinks」

  ![002](doc/002.png "002")

也可以使用命令行操作：

> git clone -c core.symlinks=true <URL>
